# Процесс развертывания

1) Запустить докер
```bash
make docker_up
```
2) Запустить Spring
```bash
make run_spring
```
Spring: <http://localhost:9001>
2) Админка

```bash
make run_front
```

Запустится на <http://localhost:3000/>

Для сборки используется Yarn!
[Установка](https://yarnpkg.com/getting-started/install)
